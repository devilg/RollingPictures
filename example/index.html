手风琴式图片效果

August 18, 2015 by nnngttdd

通过修改元素width方式实现

开始是通过鼠标时间修改元素的width来实现效果，写完之后发现当快速在图片上移动鼠标，最后一个图片元素会跑到前面，是个bug。后来又换了思路。

通过绝对定位实现

通过绝对定位的办法，通过鼠标事件，动态修改每个元素距离左侧的距离。暂时不会出现之前的问题。

配置属性

						
	var options = {
		divHeight : 200, 
		// 最外层div高度（与normalWidth二者选一）
		divWidth : 500, 
		//最外层div宽度（必填）
		normalWidth : 100, 
		// 图片正常显示大小 当divWidth定义时 无作用 （与divHeight二者选一）
		bigWidth : 310, 
		// 图片最大化显示大小（必填）
		borderWidth : 2, 
		// 图片左侧边框（必填）
		transitionTime : 2, 
		//动画移动时间（非必填）
		type : 'left' 
		//展现的方式 'left'通过绝对定位 ，'width'通过修改元素大小（非必填）
	};

	var myPictures = new RPictures('pic',options);
	//两个参数 第一个为操作的div  第二个为配置的属性
